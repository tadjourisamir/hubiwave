🌐 What is HubiWave?

HubiWave is a modular, self-hosted orchestration dashboard for local networks. Whether you're a dev, sysadmin, or automation enthusiast, HubiWave lets you control, plan, and execute actions across multiple Linux hosts — all from your browser.

⚙️ Core Features

  • 🚀 Instant SSH commands on one or multiple machines (parallel or sequential mode).
  • 📜 Script upload and execution (.sh, .py), with target selection and detachment options.
  • 📅 Advanced scheduler supporting cycles, delays, spacing, and execution plans.
  • 🧠 Persistent task storage with no database — 100% local, JSON-based.
  • 📊 Interactive calendar view for all scheduled jobs with tooltips and details.
  • 🔎 Execution logs and real-time feedback for script runs.

🧱 Architecture Overview

  • Blueprints organize features (commands, scripts, scheduler, etc.).
  • Modules are isolated and self-contained (routes, templates, static, logic).
  • Core services manage tasks, execution, metadata, and host communication.
  • Flat file storage ensures everything is transparent and portable (JSON only).

🔐 Designed for Privacy

  • 🛡️ Local-only: Nothing is ever sent to the cloud.
  • 🔑 SSH-native: No secrets are stored; your keys stay where they belong.
  • 📁 No DB: All config and data are human-readable flat files.
  • 🐧 Linux-first: Works on Linux, WSL, macOS, or any UNIX with Python 3.8+.

🚀 Get Started in 3 Steps

  1. 👥 Configure your hosts in hosts.json (or approve new ones via the Pending tab).
  2. 🖥️ Send immediate Commands or run uploaded Scripts.
  3. ⏱️ Need automation? Use the Scheduler to define recurring or timed tasks.

📦 Project Goals

HubiWave aims to be simple, auditable, offline-first, and easy to extend. Whether you're managing 2 machines or 200, it's your lightweight alternative to heavy remote management platforms.
